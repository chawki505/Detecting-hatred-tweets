#### 1- Installation des dépendances

```shell
$ pip3 install -r requirements.txt 
```

#### 2- Execution en local

```shell
$ python3 main.py
```

ou

```shell
$ python3 main.py "votre tweet de test"
```

#### 3- Execution d'un serveur web

```shell
$ python3 server.py
```
